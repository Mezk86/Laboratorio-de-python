## Descripción general del sistema

Este proyecto consiste en un sistema de venta de entradas desarrollado en Python.

El sistema permite administrar la venta de entradas para diferentes sectores de un evento, controlando:
- Sectores disponibles (VIP, Platea y General).
- Precios de cada sector.
- Cantidad de entradas disponibles.
- Registro de ventas realizadas.
- Recaudación total.
- Estadísticas de ventas.

El programa incluye validación de datos ingresados por el usuario, aplicación de promociones y almacenamiento del historial de ventas mediante archivos de texto.
